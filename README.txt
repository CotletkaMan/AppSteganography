	AppSteganography
	Программа для внедрения и распаковывания документов в WAV файлах.
Она разделяется на две области. Первая (верхняя) часть представляет режимы работы программы. Вторая часть используется для выбора требуемого алгоритма стеганографии.
Программа может выполнять четыре операции: скрытие, распаковывание, работа с конфигурационным файлом и вывод ошибок работы программы. 
	Для скрытия документа нужно перейти на вкладку Pack , указать в требуемых полях путь до контейнера,путь до скрываемого файла и место вывода работы алгоритма. Каждый выбранный документ можно проверить нажав на кнопку open. Результатом скрытия является преобразованный контейнер и конфигурационный файл использующегося алгоритма (если он используется).
	Распаковывание контейнера выполняется на вкладке Unpack. Для работы алгоритма нужно указать контейнер и место вывода программы.
	Программа позволяет работать с конфигурационными файлами методов. Для этого нужно перейти на вкладку config. Для того чтобы программа нашла настроечный файл, он должен располагаться в папке target/classes/configuration/<Название_метода>.properties
	Последним режимом работы является вывод ошибок работы программы. Для их отображения нужно перейти на вкладку log.
	Нижняя панель используется для выбора нужного алгоритма и запуска нужной операции. Нужное действие выбирается относительно активной вкладки.
