Программа делится на две части:
	1) Выбор действия для алгоритма (вкладки)
	2) Выбор самого алгоритма (на текущий момент всего один)

Для выполнения любой из вкладок нужно нажать центральную кнопку с именем метода.
Вся информация об ошибках пишется в лог. Его можно посмотреть в ссотвествующей вкладке.

Метод может содержать конфигурационный файл. Его просмотр и изменение на вкладке Config.
Результатом запаковки документа является модифицированный контейнер и конфигурационный файл при котором он был сделан.
